# 日本地理探險 | Japan Geography Quest | 日本地理探検

以日本中學・高中地理課程為基礎的互動學習 Web App，支援**繁體中文、English、日本語**三語切換。

## 功能

| 模式 | 說明 |
|------|------|
| 🗾 **探索學習** | 瀏覽 8 大地區、47 都道府縣，以及地形・氣候・災害等課本重點 |
| ❓ **地理測驗** | 20 道選擇題隨機抽 10 題，附解說 |
| 🗺️ **地圖挑戰** | 在地圖上點選都道府縣所屬地區 |
| 🃏 **閃卡記憶** | 翻卡學習都道府縣名稱、首府與趣味知識 |

## 課程內容涵蓋

- 日本列島位置與領海
- 8 大地區與 47 都道府縣
- 地形（山地、平野、河川、富士山）
- 氣候（四季、梅雨、颱風、偏西風）
- 自然災害（地震、火山、津波）
- 人口與都市（三大都市圈、少子高齡化）

## 快速開始

```bash
cd japan-geography-app
npm install
npm run dev
```

開啟 http://localhost:5173

